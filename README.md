FTE Projection Model

This project extracts case data from Salesforce and HR employee data from Workday over a defined period. The combined data is processed using an Alteryx workflow and then visualized in Tableau to provide the business with a consolidated view of past, present, and future headcount/FTE and case volume. These insights are used to create performance benchmarks and project future performance, helping to identify teams with potential staffing gaps, backlog risks, or overstaffing. This enables leaders to reallocate resources as needed to drive operational efficiency.
